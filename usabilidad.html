Que es usabilidad web?

Se refiere al nivel de facilidad a la hora de acceder y navegar por una pagina web de forma sencilla y rápida. También se trata de evitar errores comunes como imágenes que no cargan, herramientas que se queden colgadas o que se caigan, es importante además tener en cuenta al usuario a la hora de diseñar la página, intentar que sea intuitiva, segura y sencilla. Se debe tener en cuenta a la hora de diseñar una página los siguientes aspectos:

Eficiencia: Es cuando la pagina logra suplir todas las necesidades del usuario que la visita.

Eficacia: Es cuando los usuarios logran conseguir sus objetivos en el menor tiempo posible.

Satisfacción: El usuario logra una visita satisfactoria y considera que la misma fue sencilla, rápida y eficiente.

Importancia

Una buena usabilidad web repercute en muchos aspectos tales como disminución de los porcentajes de abandono, aumento de tiempo de visitas, mayores tasas de ventas cuando la página presta servicios, mejor experiencia de usuario, mayor captación de ingresos.

Principios de Jakob Nielsen:

Visibilidad del estado del sistema:

Se refiere a que la web siempre presenta la información necesaria para que el usuario sepa el estado de formularios, procesos o transacciones hechas en la página.

Relación entre el sistema y el mundo real:

Trata de utilizar, por ejemplo, palabras o frases que el usuario vea familiares o fáciles de entender, imágenes claras, iconos fáciles de reconocer, un orden lógico en su contenido.

Control y libertad del usuario:

trata de dar herramientas al usuario en caso de errores para que pueda volver a un estado anterior o deshacer una opción equivocada.

Consistencia y estándares:

Utilizar iconos estándar, optimización para dispositivos móviles, diseños responsivos, entre otros.

Prevención de errores:

es importante prevenir errores que pueda cometer el usuario con la utilización de técnicas como confirmaciones de contraseña o correos electrónicos, opciones de autocompletado, scrips de validación, etc.

Reconocer antes que recordar:

Trata de utilizar técnicas para predecir acciones, recordando antiguos formularios, por ejemplo, o utilización de previsualizaciones y editores de texto.

Flexibilidad y eficiencia de uso:

A la hora de diseñar una web hay que tener como objetivo que esta sea preparada para usuarios novatos y experimentados y con esto conseguir que cualquier persona pueda navegar en la misma. Un ejemplo seria el buscador de Google, cualquier usuario puede solo buscar contenido utilizándolo, pero usuarios mas experimentados pueden buscar información especifica utilizando los diferentes filtros y opciones de búsqueda.

Diseño estético y minimalista:

Es de suma importancia depurar constantemente los sitios web de información innecesaria, sin actualizar o no utilizada. Además no recargar el diseño de la web y buscar un sitio limpio y eficiente.

Ayudar a los usuarios a reconocer, diagnosticar y corregir los errores:

Intentar que todos los errores que se puedan presentar sean expresados en un lenguaje que los usuarios entiendan y no por códigos, ejemplo de esto es el típico “error 404”

Ayuda y documentación:

Evitar que el usuario tenga que recurrir a documentos o tutoriales externos para poder utilizar la web de manera efectiva, aun así, se debería de dar la opción de tener un pequeño manual de uso dentro de la misma pagina.
